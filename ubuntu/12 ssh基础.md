# ssh 基础

在 Liunx 中 SSH 是非常常用的工具，通过 SSH 客户端我们可以链接到运行了 SSH 服务器的远程机器上

- 数据传输是加密、压缩的，安全且快速

## 域名和端口号

- 域名是 IP 的别名
- 端口号：通过端口号可以找到计算机上运行的应用程序

常见的端口号列表

|SSh 服务器|22|
| ---- | ------ |
| 服务 | 端口号 |
|Web 服务器|80|
|HTTPS|443|
|FTP 服务器|21|

## ssh 客户端的简单使用

```
$ ssh [-p port] usr@remote
```

- usr 远程机器上的用户名
- remote 远程机器的 IP 地址或者域名
- port 是 SSH 客户端的监听端口，如果不指定就默认值 22

## scp
